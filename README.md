# lab11
Регистрируемся на сайте ngork 

Поднятие ngrok.
```
ngrok tcp 22
```
Проверка сетевого соединеня.
```
nc -v <домен> <порнт>
```
Настраиваем ssh для удалённого доступа.
```
sudo service ssh start
```
Подключаемся к нашей же машине через сеть
```
ssh <имя пользователя><домен> -p<порт>
```
